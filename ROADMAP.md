# 🚗 Raylib Car Prototype — Sprint Plan (6h/week)

---

## 📅 Sprint 1 — Velocity System (основа движения)

### 🎯 Цель:

Перейти от "позиции" к физике через velocity

### Задачи:

* [ ] Ввести `velocity` как основную переменную движения
* [ ] Перевести движение на интеграцию через velocity

### Чек-лист:

* [ ] Машина продолжает двигаться после отпускания W
* [ ] Видно замедление (friction)
* [ ] Нет "телепортации" движения

---

## 📅 Sprint 2 — Визуализация физики

### 🎯 Цель:

Начать "видеть" физику

### Задачи:

* [ ] Отрисовать forward vector
* [ ] Отрисовать velocity vector

### Чек-лист:

* [ ] Красный вектор = направление машины
* [ ] Синий = реальное движение
* [ ] При повороте векторы расходятся

---

## 📅 Sprint 3 — Axle Model (оси и колёса)

### 🎯 Цель:

Сделать геометрию машины

### Задачи:

* [ ] Добавить front/rear axle
* [ ] Добавить 4 колеса

### Чек-лист:

* [ ] Колёса всегда "приклеены" к машине
* [ ] Оси правильно поворачиваются
* [ ] Нет смещения при повороте

---

## 📅 Sprint 4 — Velocity Decomposition

### 🎯 Цель:

Разделить движение на forward и lateral

### Задачи:

* [ ] Разложить velocity на forward/lateral
* [ ] Визуализировать lateral vector

### Чек-лист:

* [ ] Есть отдельный боковой вектор
* [ ] При повороте появляется lateral
* [ ] Можно видеть "занос"

---

## 📅 Sprint 5 — Grip (сцепление)

### 🎯 Цель:

Добавить контроль над заносом

### Задачи:

* [ ] Добавить уменьшение lateral velocity
* [ ] Сделать коэффициент grip

### Чек-лист:

* [ ] Машина перестаёт скользить боком бесконечно
* [ ] При малом grip появляется дрифт
* [ ] При большом — машина "прилипает"

---

## 📅 Sprint 6 — Steering Model

### 🎯 Цель:

Сделать руление реалистичным

### Задачи:

* [ ] Поворот влияет только на переднюю ось
* [ ] Ограничить угол поворота

### Чек-лист:

* [ ] Машина не вращается на месте
* [ ] Повороты ощущаются "дугой"
* [ ] Поведение похоже на авто

---

## 📅 Sprint 7 — Basic Drift

### 🎯 Цель:

Получить управляемый занос

### Задачи:

* [ ] Ослабить lateral grip на скорости
* [ ] Добавить зависимость от скорости

### Чек-лист:

* [ ] На высокой скорости машина скользит
* [ ] Можно контррулить
* [ ] Появляется "контроль"

---

## 📅 Sprint 8 — Camera Upgrade

### 🎯 Цель:

Сделать камеру "играбельной"

### Задачи:

* [ ] Камера учитывает velocity
* [ ] Добавить lag

### Чек-лист:

* [ ] Камера отстаёт при ускорении
* [ ] Камера не дёргается
* [ ] Камера помогает видеть дрифт

---

## 📅 Sprint 9 — Track Base

### 🎯 Цель:

Создать игровое пространство

### Задачи:

* [ ] Добавить границы (box/plane limits)
* [ ] Сделать простую трассу

### Чек-лист:

* [ ] Машина не уезжает в бесконечность
* [ ] Есть визуальные границы
* [ ] Есть маршрут

---

## 📅 Sprint 10 — Checkpoints + Restart

### 🎯 Цель:

Сделать игровой цикл

### Задачи:

* [ ] Добавить чекпоинты
* [ ] Добавить reset

### Чек-лист:

* [ ] Можно "проходить трассу"
* [ ] Рестарт работает
* [ ] Есть цель

---

## 📅 Sprint 11 — Feedback (ощущения)

### 🎯 Цель:

Добавить "жизнь" машине

### Задачи:

* [ ] Визуализировать занос (линии/цвет)
* [ ] Добавить speed-based эффекты

### Чек-лист:

* [ ] Видно, когда машина скользит
* [ ] Есть ощущение скорости
* [ ] Играть стало приятнее

---

## 📅 Sprint 12 — MVP Polish

### 🎯 Цель:

Собрать минимально играбельную версию

### Задачи:

* [ ] Баланс скорости / grip
* [ ] Убрать баги

### Чек-лист:

* [ ] Можно играть 5+ минут без раздражения
* [ ] Нет критичных багов
* [ ] Управление предсказуемое

---

# 🧠 Definition of Done (для всего проекта)

* [ ] Машина управляется предсказуемо
* [ ] Есть контролируемый дрифт
* [ ] Есть трасса
* [ ] Есть базовый gameplay loop

---

# 🚀 Итог

📦 ~12 недель при 6h/week
👉 Реалистичный MVP

---

# ⚠️ Главное правило

Если спринт не зашёл:

➡️ **режь задачу пополам, а не увеличивай время**
